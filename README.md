### Хотите посмотреть установить мой проект? Следуйте инструкциям:
***
+ Клонируйте репозиторий
```  
git clone https://github.com/DChuchin/portfolio.git
```
+ Перейдите в папку с прoектом
```
cd new_folder
```
+ Установите bower
```
bower i
```
+ Установите npm
```
npm i
```
+ Запустите gulp
```
gulp
```
